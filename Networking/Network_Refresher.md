# General concepts and models

## OSI model

1. Physical - data cables, cat6
2. Data - Switching, MAC addresses
3. Network - IP addresses, routing
4. Transport - TCP/UDP
5. Session - session management
6. Presentation - WMV, JPEG, MOV
7. Application - HTTP, SMTP

## TCP
1. Network Access (Link) – Ethernet, Wi-Fi, ARP, MAC addressing
2. Internet – IP addressing, routing, ICMP, IP packets
3. Transport – TCP (reliable), UDP (fast, connectionless)
4. Application – HTTP, HTTPS, FTP, SMTP, DNS, SSH

## Ports

Ports identify services on a host:

Port	Service
80	HTTP
443	HTTPS
22	SSH
21	FTP
25	SMTP
53	DNS
